# MYAI Project

## This is a project for the CSC-3750 course at WSU.

## Project Description:

This application is designed to showcase AI stats, ranging from LLMs to CV models. Showing users the most popular and useful models that are available to them. This allows users to make sound decisions when planning their projects, understanding which models they can use, the price, and open-sourced alternatives.

## Setup

For this project you need to download [Ollama](https://ollama.com/), this enables the user to use the multi-step-form which uses a local version of llama3.2.

## Overview

### Multi-step-form

This section uses a multi-step form to help users select an AI/AI workflow given
some project information.

This section uses a **local** AI model, we use llama3.2 which we can run locally with
ollama. This is a very simple setup, all you need to do is download the ollama desktop
application.

### Leaderboard

This section showcases some of the most prominent AI models in a table format.
Allowing users to view the top models with their prices as well.

### Contact Us

This section uses **EmailJs** we can fill out a basic contact form, which allows users to send emails to the maintainer.

### Alternatives

This section shows some alternative models you can use, as well as how to set them up.
This section is in-depth and also provides resources for users.
